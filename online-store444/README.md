
# Simple Online Store (Node.js + Express + SQLite)

متجر إلكتروني بسيط متكامل (واجهات + قاعدة بيانات) يمكنك تشغيله محليًا ورفع الكود كاملًا على GitHub.

## 1. المتطلبات (Requirements)

- Node.js (الإصدار 16 أو أحدث يُفضّل)
- npm (يأتي مع Node.js)

## 2. خطوات التشغيل محليًا (Run Locally)

```bash
# تثبيت الحزم
npm install

# تشغيل السيرفر
npm start
```

ثم افتح المتصفح على:

- http://localhost:3000

## 3. ما الذي يشمله المشروع؟

- `server.js`  
  سيرفر Express مع قاعدة بيانات SQLite (ملف `shop.db` سيتم إنشاؤه تلقائيًا عند التشغيل، مع إدخال منتجات افتراضية).

- `public/index.html`  
  صفحة الرئيسية لعرض المنتجات وإضافتها إلى السلة.

- `public/cart.html`  
  صفحة سلة المشتريات (السلة محفوظة في LocalStorage في المتصفح).

- `public/admin.html`  
  صفحة بسيطة للإدارة لإضافة منتجات جديدة إلى قاعدة البيانات عبر API.

- `public/styles.css`  
  تنسيقات CSS بشكل بسيط وحديث.

- `public/main.js`  
  كود JavaScript للتعامل مع:
  - تحميل المنتجات من `/api/products`
  - إدارة السلة (إضافة/حذف/تعديل الكمية)
  - إرسال منتج جديد عبر `/api/admin/products`

## 4. ملاحظات حول GitHub

- يمكنك رفع هذا المجلد بالكامل إلى مستودع GitHub جديد.
- تذكّر أن GitHub Pages لا يشغل Node.js مباشرة؛  
  لذلك GitHub هنا سيكون لاستضافة **الكود** فقط، وليس لتشغيل السيرفر بالكامل.
- للتشغيل الفعلي تحتاج خادم Node.js (مثلاً على Render أو Railway أو VPS).
